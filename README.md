## Build & development

Run `npm install` and then `npm run dev`.

## Enviroments

Please, create file .env

```
NODE_ENV=production
GITHUB_API_URL=https://api.github.com/users
DEV_SERVER=http://localhost:3000/api/users
PORT=8080
GITHUB_USER_ID=your_user_id
GITHUB_USER_KEY=your_secret_key

```
